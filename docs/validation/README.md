## How To Read These Files

- Audit files with a date in the filename are snapshots unless the title explicitly says they are a live tracker.
- Remediation trackers capture the current state of findings, but tracked ownership and disposition still live in [../phases/planning-and-progress.md](../phases/planning-and-progress.md).
- Validation references explain current repo patterns and source files, not task ownership.
- When a task resumes from an audit or review artifact, confirm the live workboard row first and then use the relevant snapshot or tracker for context.
